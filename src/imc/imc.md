# IMC calculator

## Input

- IMC values
- IMC classification
- Weight (kg)
- Height (m)

## Output

- Print the IMC
- Print the IMC classification

## IMC classification

- Delgadez severa: < 16
- Delgadez moderada: [16 - 17)
- Delgadez leve: [17 - 18.5)
- Normal: [18.5 - 25)
- Sobrepeso: [25 - 30)
- Obesidad leve: [30 - 35)
- Obesidad moderada: [35 - 40)
- Obesidad morbida: >= 40
